# Estructura-y-Navegacion-de-Sitios-WEB-Profesionales
Aplicarás principios fundamentales de usabilidad que mejoran significativamente la experiencia del usuario y facilitan la navegación intuitiva.


## Menú de navegación persistente
El siguiente menú se muestra de forma constante en todas las páginas del sitio:

---------------------------------------------------------
| LOGO | Inicio | Tienda | Productos | Carrito | Cuenta |
---------------------------------------------------------
